BUSUR DAN LUAS JURING



A. Hubungan Sudut Pusat dengan Panjang Busur Lingkaran


Sebelum memasuki materi busur dan luas juring dengan hubungan pusat lingkarannya, mari perhatikan gambar berikut dibawah ini mengenai hubungan sudut pusat dengan panjang busur lingkaran dan luas juring lingkarannya.



Nah, dari gambar applet geogebra di atas dapat kita tahu bahwa, panjang busur adalah panjang garis lengkung yang menghubungkan dua titik pada keliling lingkaran.

Panjang busur lingkaran pada umumnya berbanding lurus dengan besar sudut yang dibentuk oleh titik pusat lingkaran dan kedua titik yang membatasi daerah garis tersebut. Menentukan Panjang busur lingkaran, sebagai berikut:

A. (Panjang busur AB) / (Keliling Lingkaran) = ( Sudut Pusat AOB ) / ( 360° ).

B. Panjang Busur AB = ( Sudut Pusat AOB ) / ( 360° ) x Keliling Lingkaran

C. Panjang Busur AB = ( Sudut Pusat AOB ) / ( 360° ) x 2 π r.


B. Hubungan Sudut Pusat dengan Luas Juring Lingkaran


Juring adalah bagian dari luas lingkaran. Dalam sebuah keluasan lingkaran yang berpusat di O dengan jari-jari dari garis OB dan akan ditarik untuk membentuk sudut tengah AOB ( OBAOB ) dengan berbagai AOB ( L.AOB ). Kemudian sudut AOB diperbesar sebagai skala AOC ( L.AOC ) yang terdapat di tengah - tengah AOC sebagai konsep nilai komparatif dari permukaan Juring -> ( ∠AOB ) / ( L.AOC ) = ( L.AOB ) / ( L.AOC ). 

Dalam sebuah sudut tengah AOB dengan area AOB Juring ditingkatkan ke sudut tengah AOD dengan area AOD Juring yang memiliki bentuk yaitu

🠞 OBAOB / ( ∠AOD ) = ( L . AOB ) / ( L . AOD )



Luas juring pada suatu lingkaran merupakan bagian dari luas lingkaran. Cara menghitung luas juring diperoleh melalui besar sudut yang dibentuk per 360° (jumlah sudut lingkaran) dikali luas lingkaran. Sedangkan luas segitiga dapat dihitung melalui rumus LΔ = 1 / 2 x a x t, di mana :

a = alas segitiga dan

t = tinggi segitiga.

Ketika semakin besar sudut pusatnya, maka semakin besar pula luas juring yang akan terbentuk.

Menentukan Luas Juring Lingkaran, sebagai berikut :

A. ( Luas Juring AOB ) / ( Luas Lingkaran ) = ( Sudut Pusat AOB ) / ( 360° ).

B. Luas Juring AOB = ( Sudut Pusat AOB ) / ( 360° ) x Luas Lingkaran.

C. Luas Juring AOB = (Sudut Pusat AOB ) / ( 360° ) x πr^2.

Perhatikan applet geogebra berikut dibawah ini!



Berdasarkan gambar pada applet geogebra di atas, terlihat bahwa :

Besar ∠AOC = 90°, dan Panjang busur AC = 1 / 4 Keliling lingkaran.

🟆    Rumus :

🠞     ( Panjang Busur AC ) / ( Panjang Busur BD ) = ( ∠AOC ) / ( ∠BOD )

🠞     Panjang Busur BD = ( ∠BOD ) / ( 360° ) x keliling lingkaran

(Agus, 2008) mengatakan bahwa semakin besar sudut pusat maka akan semakin besar pula panjang busur dan semakin besar juga luas juringnya. Sedangkan saebaliknya, semakin kecil sudut pusat maka akan semakin kecil panjang busur dan semakin kecil juga luas juringnya.

🠞    ( Luas Juring AOC ) / ( Luas Juring BOD ) = ( ∠AOC ) / ( ∠BOD )

🠞      Luas Juring AOC = ( ∠AOC ) / ( 360° ) x luas lingkaran


LATIHAN


1) Pada gambar applet geogebra dibawah ini, coba tentukan Panjang busur lingkaran jika diketahui sudut pusatnya adalah 30° dan jari-jarinya 21 cm!



Penyelesaian :

Panjang Busur = ( Sudut Pusat ) / ( 360° ) x 2 . π . r

r = Panjang O ke B

Sudut Pusat = O

⓵ Pertama, hitung satu lingkaran penuhnya

🟆    Rumus keliling lingkaran: 2 . π . r

🠞    : 2 x 22 / 7 x 21

🠞    : 44 x 3 = 132 cm

Maka :

🠞    Panjang Busur = ( Sudut Pusat ) / ( 360° ) x 2 . π . r

🠞    Panjang Busur = ( 30° ) / ( 360° ) x 132

🠞    Panjang Busur = 1 / 12 x 132

🠞    Panjang Busur = 11 cm

Jadi, Panjang busur lingkaran tersebut adalah 11 cm.

2) Berdasarkan gambar applet geogebra dibawah ini berapakah Luas Juring lingkaran dengan jari-jarinya 21 cm dengan sudut pusatnya adalah 60°?



Penyelesaian :

🠞     Luas = π . r^2

🠞    = 22 / ( 7 ) x 21 x 21

🠞     = 22 x 63

🠞    = 1386 〖cm〗^2

✶    Luas Juring = ( Sudut Pusat ) / ( 360° ) x π . r^2

🠞    Luas Juring = ( 60° ) / ( 360° ) x 1386

🠞    Luas Juring = 1 / 6 x 1386

🠞    Luas Juring = 231 〖cm〗^2

Jadi,luas juring lingkaran tersebut adalah 231 cm^2

3) Diketahui Panjang busur suatu lingkaran adalah 16,5 cm. Jika Panjang diameter lingkaran tersebut adalah 42 cm, maka berapa ukuran sudut pusatnya? ( π = 22 / 7)

Penyelesaian :

Karena yang diketahui adalah diameter maka kita menggunakan rumus:

🠞     Panjang Busur = ( Sudut Pusat ) / ( 360° ) x π . d

~ Misal, Sudut pusat : a

🠞     16,5 = a / ( 360° ) x 22 / 7 x 42

🠞     16,5 = a / ( 360° ) x 22 x 6

🠞     16,5 = a / ( 60° ) x 22

🠞     16,5 x 60 = a x 22

🠞     990 / 22 = a

🠞      45 = a

Jadi, besar ukuran sudut pusatnya adalah 45°.
